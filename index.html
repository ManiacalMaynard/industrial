Simple Timer Project

A genuinely fun little one page app (SPA) to build over two lunch breaks as part of an online challenge. Asked a bunch of friends to beta test it and actually found a fault! The play button, if activated again, causes the timer function to run multiple instances. Since the "time" is just a counter of how often the timer function repeats, this causes all sorts of havoc!
